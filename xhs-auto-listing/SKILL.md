# 小红书店铺一键铺货

本技能用于提供一个“可直接照着点”的完整操作流：以「小红书一键铺货」为目标，输出从打开 App 到发布成功的逐步操作。

## 适用场景

- 用户说：“将今天热门商品上架到我的小红书店铺”类似的指令
- 用户要求“每一步点哪里/输入什么/看到什么”的完整流程

## 前置条件（默认假设）

- 已安装并登录1688AI版，小红书千帆
- 手机已联网

若用户未满足前置条件，本技能仍应给出对应分支步骤。

## 目标

- 在《小红书》中上架今天新的热门商品，并确认发布成功。

## 操作步骤（Android 适用）

1. 打开1688AI版
   - 页面/位置：手机桌面
   - 操作：点击「1688AI版」图标
   - 你会看到：1688AI版主界面（顶部常见有「对话 / 商机」2个入口）

2. 进入「商机」
   - 页面/位置：顶部导航栏
   - 操作：点击「社媒」
   - 你会看到：小红书社媒近3天热度商机列表

3. 进入「商品详情页」
   - 页面/位置：商品列表
   - 操作：点击「商品」
   - 你会看到：商品详情页（下滑可以看到「立即铺货 / 立即下单」按钮）

4. 进入「铺货页」
   - 页面/位置：底部文字按钮
   - 操作：点击「立即铺货」按钮
   - 你会看到：已经绑定的店铺（下方有「铺货设置 / 立即铺货」按钮）

5. 铺货操作
   - 页面/位置：发布入口弹窗/页面
   - 操作：点击「立即铺货」按钮
   - 你会看到：「铺货成功」提示（有「铺货日志」按钮）

6. 进入铺货日志 
   - 页面/位置：弹框的「铺货日志」按钮
   - 操作：点击「铺货日志」按钮
   - 你会看到：铺货记录页面（包含已经铺货的商品列表）

7. 上架商品
   - 页面/位置：铺货记录页面
   - 操作：点击左侧复选框，选中刚刚铺货的商品
   - 操作：点击「更多」按钮
   - 操作：点击「上架」按钮
   - 操作：点击「确认」按钮
   - 你会看到：上架成功的提示

8. 确认商品是否上架成功
   8.1 打开小红书千帆APP
   - 页面/位置：手机桌面
   - 操作：点击「小红书千帆」图标
   - 操作：点击「全部商品」图标
   - 你会看到：商品管理页面（包括「售卖中 / 已下架 / 整改 / 审核中」等多个Tab）

   8.2 确认商品是否上架成功
   - 页面/位置：售卖中Tab
   - 确认信息：如果售卖中Tab下面的商品列表包含刚刚上架的商品，则整个流程成功完成

9. 失败重试
   - 触发场景：在售卖中Tab找不到刚刚上架商品的情况下触发
   - 重复上述 1-8 操作步骤，直到商品成功上架。


## 常见卡点与排障（给出可执行替代路径）

1. 上架商品时选择不了左侧复选框
   - 方案 ：需要等待商品复制成功再重试

## 完成标志

- 在小红书千帆APP售卖中Tab能够找到刚刚上架的商品
